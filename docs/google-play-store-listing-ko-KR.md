# Google Play 스토어 등록정보 — 한국어(ko-KR)

기준일: 2026-06-19

## 입력 문구

### 앱 이름

별 강화하기

### 간단한 설명

운석 파편으로 별을 강화해 특이점에 도전하는 우주 성장 게임

### 자세한 설명

작은 운석 조각에서 시작해 행성, 항성, 블랙홀을 지나 특이점까지 도전하세요.

**별 강화하기**는 재료를 선택하고 강화 버튼을 눌러 더 높은 단계에 도전하는 우주 테마 성장 게임입니다. 성공하면 별은 더 강하고 화려한 모습으로 성장하지만, 고강화 구간에서는 실패와 소멸의 위험도 커집니다.

## 게임 특징

- 0강부터 30강 특이점까지 이어지는 강화 여정
- 운석 파편, 별의 조각, 온전한 별핵 조각, 특이성 조각, 원초의 별 등 5종 재료
- 재료마다 달라지는 성공률과 사용 가능 구간
- 일반 성공, 대성공, 실패, 균열, 소멸이 만드는 긴장감
- 소멸 후 재도전하거나 부활로 안전한 단계에서 이어 가는 선택
- 행성 모양과 강화 기록을 모으는 도감
- 별 채굴 미니게임과 선택형 보상형 광고

가장 높은 단계에 도달할수록 한 번의 선택이 더 중요해집니다. 재료를 모으고, 성공 확률을 확인하고, 나만의 별을 특이점까지 성장시켜 보세요.

※ 만 13세 이상 이용가입니다. 보상형 광고는 사용자가 직접 선택한 경우에만 표시됩니다.
